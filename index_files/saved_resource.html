This visualization helps us understand the composition of the International Futures Model. The dots represent variables while the boxes represent parameters. The sizes of the variables and the parameters are determined on the basis of the number of connections these variables and parameters share with the rest of the model.

The user can make use of multiple options. The Select View option enables the user to select either a Quilt View which provides an overview of the different systems in IFs, The sub-model view that enables a user to view interactions between sub-models and the Individual Model View which enables the user to look at components of individual models. Please click on the Store positions option at the bottom to stabilize the visualization.
